# Cadastro de alunos

## 📌 Sobre o Projeto

O sistema consiste em:

- ✅ Uso de modularização para um código mais organizado
- ✅ Uso de switchs para menus
- ✅ Estruturas de repetição
- ✅ Exibição de média
- ✅ Cadastro de alunos com nome e matrícula e notas
- ✅ Visualização de boletim
---
O objetivo do projeto é praticar praticar C e melhorar.
